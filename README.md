# The-algorithm-society-python-quests 

This is a repository for our group assignment

## Project structure 

All scripts are located in the quests directory

The goal of these quests is to build your foundation in programming. Ensure to have Python present on your system via the terminal with command python --version or python3 --version

TO RUN 
Use python3 <name of the python file >
Grant executable permission by chmod u+x < name of the file > if the file is not having an executable permission
